透過 Language Wrapper 打包模型映像檔部署

Enterprise Applicable to Enterprise tier only

此教學將部署已透過 language wrapper 打包的模型映像檔。我們以已打包的 TensorFlow2 MNIST 模型映像檔為範例。

前置條件

開啟工作群組的 Model Deployment 功能

須先開啟工作群組的 Model Deployment 功能；請洽系統管理者。

教學步驟

  1. 登入 User Portal 並選擇 Models。

  2. 在 model deployment 主頁 點擊 Create Deployment。

  3. Deployment name 欄位中填入 quickstart-mnist：

    Model Image 欄位中填入 infuseai/model-tensorflow2-mnist:v0.2.0； 此已打包模型映像檔由 InfuseAI 提供並發佈於 Docker Hub。

  4. Resources 選項：

    • 選擇 instance type，我們選擇 (CPU: 0.5 / Memory: 1 G / GPU: 0)作為範例
    • 保留 Replicas 預設值 (1)

  5. 點擊 Deploy 後，導轉回主頁。等待同時，可以點擊 Refresh 更新部署狀態。

    部署成功後(綠色)，點擊該部署查看詳細資訊。
