The Environment Variables for MinIO

EnvironmentDescriptionOptional
PRIMEHUB_STORE_ACCESS_KEYThe access key of MinIONo
PRIMEHUB_STORE_SECRET_KEYThe secret key of MinIONo
MINIO_S3GATEWAY_ACCESS_KEYAWS S3 access key for MinIO gatewayYes
MINIO_S3GATEWAY_SECRET_KEYAWS S3 secret key for MinIO gatewayYes
MINIO_GCSGATEWAY_PROJECT_IDThe project id of the Google Cloud StorageYes
MINIO_GCSGATEWAY_GCS_KEY_JSONThe key file of the service accountYes
  • The s3 gateway would enable if both MINIO_S3GATEWAY_ACCESS_KEY and MINIO_S3GATEWAY_SECRET_KEY were set.
  • The gcs gateway would enable if both MINIO_GCSGATEWAY_PROJECT_ID and MINIO_GCSGATEWAY_GCS_KEY_JSON were set.
  • If both s3 and gcs were enabled, s3 would be enabled.
