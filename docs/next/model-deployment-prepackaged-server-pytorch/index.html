Model URI Structure

<model uri>
├── ModelClass.py
└── model.pt

There should be two files in the model URI

  1. model.pt: The model file should be saved by state_dict() format.
  2. ModelClass.py: A python defines the PyTorchModel class.

For more information, please see PyTorch Saving & Loading documents

How It Works

You can check the detailed code in Github. Here, we demonstrate by the pseudo-code.

Load the model

def __init__(self, model_uri):
    model_file, model_class_file = load_from_model_uri(model_uri)
    model_class = load_model_class(model_class_file)
    self.model = model_class()
    self.model.load_state_dict(torch.load(model_file))
    self.model.eval()

Predict

def predict(self, X):
    with torch.no_grad():
        inputs = torch.from_numpy(X)
        return self.model(inputs).numpy()
