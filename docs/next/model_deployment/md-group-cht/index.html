Group Management

Deploy Applicable to Deploy Edition

Groups List

Creating New Group

點選 Add 新增 Group，會跳出該 Group 的編輯畫面。

需填入以上畫面中的各個欄位：

  • Name必填，只能填寫小寫字母、數字、連接號（“-”）和底線（“_”）。

  • Display name

Group Quota

設定 Group 的總配額

  • CPU Quota 群組 CPU 總配額。預設值為沒有限制。

  • GPU Quota 群組 GPU 總配額。預設值為沒有限制。

  • Memory Quota 群組 Memory 總配額。預設值為沒有限制。

點選 confirm 完成新增。

Users

列出此群組所有使用者，並可透過edit users編輯，加入/移除使用者。

Deleting Group

點選 Actions 的Delete，會跳出確認對話框， 點選 OK 後即可刪除該 Group 。

Editing Group

點選 Actions 的Edit 進該 Group 的編輯頁面。
