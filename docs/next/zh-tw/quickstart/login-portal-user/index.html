User Portal

這份文件說明介紹使用者層級的User Portal。

登入

首先選擇 language及用「使用者」帳號密碼登入。

Portal

Portal 頁面上的捷徑圖示會因實際環境而有差異，基本上會有：

  • JupyterHub

    點擊後導向JupyterHub啟始前置頁面。使用者可以由此進行啟始專案。

  • User Guide

    點擊後導向Documentation。使用者可以由此參考 PrimeHub 相關文件。

登出

點擊右上角的Logout。 導回登入頁
