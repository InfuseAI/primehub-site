加大 User Volume

Enterprise Applicable to Enterprise tier only

這份文件介紹如何利用 Maintenance Notebook 來增加 User Volume Size。

請注意： 目前 User Volume 只能增加容量，不能減少；一旦加大，無法還原。 PrimeHub v2.0以上才有Resize User Volume功能

步驟

  1. 以管理者帳號登入後切換至 Admin Portal，進入 Maintenance Notebook。

  2. 至Resize User Volume段落，執行第一個cell後，會顯示User vol:下來選單。

  3. 從User vol:選單，選擇要變更的使用者。

  4. 執行第二個cell，顯示Current Size及New Size:輸入欄位。

  5. 輸入期望的size後 (必須大於 current size)，按下Update更新。

  6. 一旦更新完成後，可以再次執行第二個cell做為驗證。
