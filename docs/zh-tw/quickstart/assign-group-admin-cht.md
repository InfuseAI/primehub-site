這份文件說明如何為群組指定群組管理者。


1. 以管理者帳號登入後，切換至 Admin Portal。

2. 點擊 `Groups`進入群組管理介面後，選擇群組並編輯。

3. 至 Members 從現有成員中勾選(Group Admin)群組管理者(可複選) 或 點擊 `edit users` 新增成員並勾選。
    ![](assets/group_admin.png)

4. 點擊 Confirm.
