建立使用者

這份文件說明如何建立使用者。

  1. 以管理者帳號登入User Portal 後，點擊Admin Dashboard進入管理介面。

  2. 在側邊選單上，點擊 Users進入使用者管理介面後，點擊+ Add按鈕，建立使用者。

  3. 輸入使用者必要資訊，如需要 email 認證，啟用activation email。

  4. 點擊 Confirm.
